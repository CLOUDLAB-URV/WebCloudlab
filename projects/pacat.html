PACAT

Platform for Conversational Agents with Aina Kit

  • Start date: 2025-09-01
  • End date: 2025-12-31
  • Funded by: Generalitat de Catalunya through the Aina CHALLENGE
  • Ref:

The project involves developing an accessible platform for creating personalized conversational agents from documents or web pages, using Retrieval Augmented Generation (RAG) techniques and with native Catalan support via the Aina Kit. Initially, it will be validated in a university setting, allowing students and professors to interact with agents specialized in this field.
